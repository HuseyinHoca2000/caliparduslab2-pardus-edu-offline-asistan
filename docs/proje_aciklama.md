# Proje Açıklaması

Bu proje, Pardus işletim sistemi kullanıcılarına temel rehberlik sunabilen çevrimdışı çalışan bir masaüstü asistan geliştirmek amacıyla hazırlanmıştır.

Yeni kullanıcılar, özellikle Pardus ortamına ilk geçiş sürecinde sistem ayarları, uygulama kurulumu ve temel kullanım senaryolarında desteğe ihtiyaç duyabilmektedir.

Bu proje kapsamında geliştirilen offline asistan:

- kullanıcıyı karşılar
- temel soruları yorumlar
- kısa cevaplar üretir
- internet bağlantısı olmadan çalışır

## Amaç

- Pardus kullanımını kolaylaştırmak
- Yeni kullanıcı adaptasyonunu hızlandırmak
- Eğitim kurumları için yardımcı araç geliştirmek
- Offline çalışan hafif bir asistan sunmak

## İlk Sürüm Özellikleri

- Metin tabanlı sohbet
- Soru-cevap sistemi
- Basit komut analizi
- Terminal tabanlı kullanım

## Gelecek Geliştirmeler

- Sesli komut desteği
- GUI masaüstü widget
- Yerel mini LLM entegrasyonu
- Sistem komutlarını çalıştırabilme
