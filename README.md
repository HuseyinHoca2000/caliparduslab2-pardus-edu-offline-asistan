# ÇalıPardusLab2 Pardus EDU Offline Asistan

Bu proje, Pardus işletim sistemi üzerinde öğrenciler ve öğretmenler için çevrimdışı çalışabilen hafif bir masaüstü yardımcı asistan geliştirmeyi amaçlamaktadır.

## Proje Amacı

Pardus kullanan yeni kullanıcılar, öğrenciler ve öğretmenler bazı temel işlemleri gerçekleştirmekte zorlanabilmektedir. Bu proje ile kullanıcıya rehberlik eden, temel sorulara cevap verebilen ve sık kullanılan işlemlere yardımcı olan bir offline asistan geliştirilmesi hedeflenmektedir.

## Hedefler

- Yeni kullanıcıların Pardus’a adaptasyonunu kolaylaştırmak
- Temel sistem işlemlerinde rehberlik sunmak
- Eğitim ortamlarına uygun yardımcı asistan geliştirmek
- İnternet olmadan çalışabilen hafif bir çözüm sunmak

## Temel Özellikler

- Kullanıcıyı karşılama
- Sık sorulan sorulara cevap verme
- Basit komut analizi
- Sistem rehberliği
- Offline çalışma

## Hedef Kullanıcılar

- Öğrenciler
- Öğretmenler
- Pardus’a yeni geçen kullanıcılar
- Eğitim kurumları

## Kullanılan Teknolojiler

- Python 3

## Çalıştırma

```bash
python3 src/pardus_edu_asistan.py
```
